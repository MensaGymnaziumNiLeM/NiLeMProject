Instructions

This work provides You with lessons. If You want to learn something, welcome!

To return to the main menu, use the button above.

Main menu gives You the possibility to change the languages and to choose the subject.

The big buttons with labels are used to choose the subject, click on one of them.

You will get to the subject menu. Subject menu may have some lessons.

Button = lesson. No buttons = no lessons.

Click on a lesson button. You will get to the lesson.

It gives You some explanation, it gives You some questions to be answered.

Use this to move forward.

Back to Math menu Use this to exit to the subject menu.

Back to Main menu Use this to exit to the main menu.

Enjoy!
