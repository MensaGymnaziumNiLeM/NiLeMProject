Anleitung

Diese Arbeit bietet Ihnen Lektionen. Wenn Sie etwas lernen möchten, willkommen!

Um zum Hauptmenü zurückzukehren, verwenden Sie die Schaltfläche oben.

Hauptmenü gibt Ihnen die Möglichkeit, die Sprachen zu ändern und das Thema auszuwählen.

Die großen Schaltflächen mit Beschriftungen werden verwendet, um das Thema auszuwählen, klicken Sie auf eine davon.

Sie gelangen in das Themenmenü. Das Themenmenü enthält möglicherweise einige Lektionen.

Schaltfläche = Lektion. Keine Knöpfe = kein Unterricht.

Klicken Sie auf eine Unterrichtsschaltfläche. Sie gelangen zur Lektion.

Es gibt dir einige Erklärungen, es gibt dir einige zu beantwortende Fragen.

Verwenden Sie dies, um fortzufahren.

Zurück zum Mathe-Menü Verwenden Sie dies, um zum Themenmenü zurückzukehren.

Zurück zum Hauptmenü Verwenden Sie dies, um zum Hauptmenü zurückzukehren.

Viel Spaß!
